코딩은 세상과 소통하는 방식 중 하나입니다.

웹사이트란 온라인 미디어 중 단연 가장 노출이 높으며 소통을 활발히 할 수 있는 기회를 제공하죠.

스터디버디 웹개발 2달 완성반은 말그대로 2달만에 웹사이트의 모든 것을 배우는 수업입니다.

수업 대상: 컴퓨터, 웹 기술은 끈임없이 발전하고 있어 그 모든 기술을 배우는 건 불가능해요. 시간이 지나면 새로운 기술이 등장하기 때문인데요. 그렇기 때문에 기본기가 더더욱 중요해요. 이 수업은 코딩을 몰라도, 잘 해도 도움이 되는 수업이니 누구나 들어도 됩니다. 강의 진행 속도는 누구나 즐겨 들을 수 있는 수준이며, 프로젝트 진행 속도도 본인의 속도에 맞추어 진행할 수 있습니다.
수업 목표: 이 수업을 들은 수강생은 수업이 완료됨과 동시에 두개의 웹사이트를 실제로 개발하였을 것이고, 웹사이트의 기본이 되는 HTML, CSS, 자바스크립트를 자유자재로 다룰 수 있게 될 겁니다. 추가로 프레임워크의 사용은 물론, 프레임워크가 없어도 스스로 힘으로 세상에 자신이 개발한 웹사이트를 공유할 수 있는 능력을 키우게 될 겁니다.
수업 방식: 온라인 Zoom 화상회의 (오프라인 수업 병행 상담 후 결정 가능)
수업 자료: 자체 제작 자료, 깃허브에서 소스코드 제공
[월수] 온라인 쇼핑몰 개발반
제가 요즘 그릭 요거트를 만드는데 푹 빠졌는데요~ 실제로 저와 함께 2달 동안 그릭요거트를 판매하는 예쁜 온라인 쇼핑몰을 만들고 싶은 분은 이 수업을 들으면 큰 도움이 될 거에요. 제가 10년을 개발하면서 터득한 모든 노하우를 여러분께 상세하게 알려드리고 제 수업을 들으면서 직접 쇼핑몰 운영을 할 수 있도록 코드 한줄한줄 설명드릴게요. 이 수업 수강생분들은 그릭 요거트 100원에 살 수 있어요~ :)

1주차: HTML, CSS 이론/실습 - 첫 "정적" 웹사이트 제작

2주차: 자바스크립트 A to Z (Frontend) - 첫 "동적" 웹사이트 제작

3주차: 자바스크립트 응용 (Backend) + REST API 제작

4주차: 데이터베이스 + 리눅스 웹배포

5주차: 반응형 쇼핑몰 페이지 제작 (판매, 진열, 고객, 장바구니 페이지)

6주차: 고객 관리 시스템 제작 (로그인, 소셜 로그인)

7주차: 홍보 페이지 / 홍보 영상 업로드 + FAQ/공지 게시판 페이지

8주차: 마무리 단계 - 웹사이트 배포, 도메인 등록, 서버 관리 등

[토일] 금융 데이터 대시보드반
제 첫 개발 프로젝트가 DART에서 국내 전종목 재무 데이터를 긁어오는 거였는데요. 그렇게 키스톤 인베스트먼트에서 로보 어드바이저 개발부터 자동 매매 시스템까지 수년간 금융 데이터를 핸들링하였어요. 금융은 제 인생이고, 제 미래이기도 하죠. 금융 데이터는 분석과 시각화를 통하여 좋은 인사이트를 얻어낼 수 있어요. 저와 함께 국내 3000개, 해외 49000개 종목 가격, 재무 데이터를 분석해봐요. (수강생들은 20년치 데이터 무료 제공)

1주차: HTML, CSS 이론/실습 - 첫 "정적" 웹사이트 제작

2주차: 자바스크립트 A to Z (Frontend) - 첫 "동적" 웹사이트 제작

3주차: 파이썬 A to Z (Backend) + REST API 제작

4주차: 데이터베이스 + 리눅스 웹배포

5주차: Django로 국내(3000개 종목)/해외(49000개 종목) 가격/재무 데이터 수집 - 데이터 전부 제공

6주차: 반응형 검색 페이지 제작

7주차: 데이터 분석과 시각화의 기본 (Numpy, Pandas, Highcharts.js)

8주차: 마무리 단계 - 웹사이트 배포, 도메인 등록, 서버 관리 등
